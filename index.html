Loud Vibes

Loud Vibes is a small reggae and dancehall concert site designed to share upcoming concerts, reviews, and music news-while keeping gossip to a minimum.

I started by creating user personas for Loud Vibes target users. The target users are true fans of reggae and dancehall and use it as their outlet from their very hectic lives. This led me to pay more attention to the mobile experience. I designed mobile first and adjusted the layout to the devices using progressive enhancement. I relied on the user flows to develop a sitemap that made it simple for users to complete vital tasks. After testing my site, I adjusted the layout to create the best design solution for Loud Vibes users.

This is the first project I created a style tile for instead of a mood board and I am really excited about how functional they are when it is time to code. All of my design elements are availablehere.

aleiawalker.com

This is the third stylistic iteration of a site which started as a capstone project for a web design course. I started by building out the user personas and found the target user to be hiring managers with short amounts of time to dedicate to prospective employee portfolios. That is why I concentrated on a single page layout so that everything was immediately accessible to the user. After the user personas I built out user flows, sitemaps, and wireframes—all codependent on one another. I created all of the design elements (logo, logo mark, and signature) and incorporated them into my mockup. The site was not designed mobile first but I will be redesigning it to value the mobile experience.

In my mobile-first redesign, I plan to convert the HTML to HTML5, the CSS to SASS, and use Flexbox to stabilize the layout. I will also use the jQuery flyout menu that I created for the mobile version to incorporate a hidden menu into my new design. Please check out some of the previous versions ongithub.
